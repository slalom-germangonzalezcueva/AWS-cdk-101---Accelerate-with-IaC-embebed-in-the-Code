# Demo Scope

## What is this about?

We are going to deploy a simple application to maintain my own <i>curriculm vitae</i>

### What is included in the demo?

S3, CloudFront, API Gateway, Lambda, DynamoDB, React frontend, GitHub OIDC

### What is NOT inlcuded?

Authentication, multi-user tenancy, VPCs, load balancers.

> Lets not get crazy with the budget.
